# Examples

`CLAUDE.md.template` の記入例集です。コピーして自社情報に書き換えてご利用ください。

## ファイル一覧

| ファイル | 内容 |
|---|---|
| [CLAUDE.sample-studio.md](./CLAUDE.sample-studio.md) | 一人合同会社で SaaS + 受託を運営している想定の記入例。`{{...}}` プレースホルダーをすべて埋めた状態のサンプル。 |

## 使い方

```bash
# サンプルをベースに自社の CLAUDE.md を作る場合
cp examples/CLAUDE.sample-studio.md /path/to/your/project/CLAUDE.md
# その後、自社情報に書き換える
```

実例を見ながら自社用に書き換えるほうが、`{{...}}` プレースホルダーをゼロから埋めるより速いケースが多いです。
