## 事業フェーズ

## プロダクト一覧

### （自主開発）TaskBuddy（試作中）
- 概要：自分の請求書発行ワークフローを SaaS 化する個人開発プロジェクト
- ターゲット：自分と同じ属性のフリーランス・個人事業主
- 主機能：請求書テンプレート・freee 連携・税込/税抜の自動切替

## 技術スタック
- フロントエンド：React + TypeScript + Tailwind CSS（受託・自主開発とも）
- バックエンド：Node.js + Express、もしくは Hono

- 外部 API：Stripe、freee API
- インフラ：Vercel（自主開発）、AWS（受託案件で頻出）

## 行動・判断の必須ルール
1. 個人事業主前提で考える。法人化提案は財務メリットが明確なときのみ
2. 受託案件の納期・品質を最優先しつつ、週 5 時間は自主開発に確保する前提で助言する

4. 国保・年金など個人事業主特有の社会保険コストを常に考慮した手取りベースで判断
5. 回答は日本語で行う。コード内コメント・変数名は英語可
6. 健康・労働時間の制約（過労リスク）を提案に必ず織り込む
